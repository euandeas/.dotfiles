# .dotfiles

My personal dotfiles configuration. Designed for use with Fedora Minimal Install & [DMS](https://danklinux.com/).

## Installation

1. Setup Fedora & DMS
2. Clone this repository to your home directory:
```bash
cd ~
git clone https://github.com/euandeas/.dotfiles
cd .dotfiles
./install.sh
```

For installtion script details click [here](install/README.md)
